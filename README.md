# 4th Monthsary — Jea

A single-page HTML site. Just open `index.html`, no build step needed.

## Deploy to Vercel

1. Push this folder to a GitHub repo.
2. Go to https://vercel.com/new and import the repo.
3. Framework preset: "Other" (or leave as detected). No build command needed — it's a static `index.html`.
4. Deploy.
